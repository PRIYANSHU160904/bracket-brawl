# BracketBrawl

A real-time competitive programming duel platform where two players
compete to solve the same problem.

## Features
- Real-time matchmaking using WebSockets
- Room-based gameplay
- Code execution and evaluation using Piston API

## Tech Stack

- React
- Node.js
- Express
- Socket.IO

## Project Structure

- client/ → Frontend
- server/ → Backend

## Status

This project is currently under development.
